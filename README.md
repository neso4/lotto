# 미션 - 로또

## 🚀 기능 구현
- [x] 로또 발행
- [x] 당첨 번호 추첨
- [x] 당첨 번호 + 보너스 번호 입력
- [x] 당첨 내역 출력
- [x] 수익률 출력
- [x] 예외 상황 출력 
  - 범위가 일치하는지 ✅
  - 보너스 번호가 입력한 당첨번호와 중복되지 않는지 ✅
  - 입력한 당첨번호끼리 중복하지 않는지 ✅
  - 숫자만 입력했는지 ✅
  - 개수 맞는지 ✅
  - 정수인지 ✅
