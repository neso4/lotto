# 기능 목록
### 1. 로또 구입 금액 입력
- 입력 받은 값의 예외 처리
  - 양수인 정수가 아닌 값
  - 1000원 단위가 아닌 값
### 2. 구입한 만큼의 로또를 발행
### 3. 당첨 번호와 보너스 번호를 입력
- 당첨 번호 입력 받는 기능
- 보너스 번호 입력 받는 기능
- 입력 받은 값의 예외 처리
  - 1~45 사이의 정수가 아닌 값
  - 당첨 번호나 보너스 번호가 중복 되는 값
  - 당첨 번호가 6개가 아닌 값
### 4. 등수별 당첨 장 수 출력
### 5. 수익률 출력
- 수익률 = (수익 / 로또 구입 금액) * 100
  - 소수점 둘째 자리에서 반 올림
