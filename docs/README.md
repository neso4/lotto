2. 로또길이, 중복된 숫자 있는지 체크(예외처리)

[Control Class]
1. 로또 구매 가능 개수 체크, 천원 단위 체크(예외처리)
2. 구입금액만큼 난수 생성 및 로또 객체 생성
3. 정렬되어있는지 체크(예외처리)
4. 입력된 당첨 번호 예외처리(중복체크, 보너스 번호 중복체크)
5. 로또 당첨 계산
6. 수익률 계산

[View Class]
1. 프로그램 시작 메시지 출력
2. 각종 필요한 수 입력
3. 결과 출력
