# 기능 목록

## 로또
1. 1장에 1,000원이다.
2. 서로 다른 6개의 번호를 가진다.
   1. 번호는 오름차순으로 정렬한다.

## 구매자
1. 구입 금액을 입력 한다.
   1. 1,000원 단위가 아니면 안내 문구를 출력하고 재입력 받는다.
2. 구매한 로또를 수령 한다.

## 로또 발행 기계
1. 로또를 발행 한다.
2. 발행한 수량을 출력 한다.

## 관리자
1. 당첨 번호를 입력 한다.
   1. 번호는 쉼표를 기준으로 구분 한다.
2.  보너스 번호를 입력 한다.

## 컴퓨터
1. 관리자 당첨 번호 입력 유효성 검증
   1. 예외는 "[ERROR]" 로 시작 한다.
2. 구매자 로또와 당첨 번호를 비교 계산 한다.
3. 2에서 계산한 값을 출력 한다.
   1. 수익률은 소수점 둘째 자리 반올림
