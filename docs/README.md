# @입력

Cost

- 설명 : 로또를 사기위해 구매할 돈
- 조건 : 구매할 로또는 한장에 천원, 따라서 천원 단위가 아니면 Error

winNum

- 설명 : 당첨 로또 번호
- 조건 : 6자리여야 하고 중복되면 안됨

bonus

- 설명 : 보너스 번호 ( 로또 번호랑은 다름 ) - 로또번호 5개 + 보너스 번호 당첨이 있음
- 조건 : 당첨번호에는 없는 숫자여야함

# @출력

    구입금액을 입력해 주세요.
- Cost 입력 받음


    N개를 구매했습니다.
- Cost/1000 = N


    6 * N 생성된 로또번호 출력
- 중복 없고, 오름차순


    당첨 결과 화면
    통계 화면
