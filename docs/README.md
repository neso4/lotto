## 기능 구분
    * 메인 클래스
        1. 로또 구입 금액을 입력받는다.
        2. 금액에 따른 장 수만큼의 번호를 출력한다.
            a. 로또 클래스 이용
        3. 보너스 번호를 입력받는다.
            a. 
        4. 당첨 번호를 출력한다.
            a. 로또 클래스 이용
        5. 당첨 통계를 출력한다.
            a. 구입한 로또 번호화 당첨 번호가 일치하는지 검사한다.
            b. 일치한 개수의 당첨 내역을 출력한다.
            c. 수익률을 출력한다.

    * 로토 클래스
        1. Integer 리스트 numbers를 인자로 받는다.
        2. 해당 numbers가 유효한지 검증하다.

    * 로또 번호 생성 클래스
        1. 로또 구입 금액에 따라 번호를 생성한다. 
        2. 로또 당첨 번호를 생성한다. 

    * 로또 구입 금액 검증 클래스
        1. 로또 금액이 1000원 단위로 떨어지는지 검증한다.

    * 번호 일치 검증 클래스
        1. 일치한 개수에 따라 분류한다.
        2. 총 수익률을 계산한다.
