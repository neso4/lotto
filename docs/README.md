# 🔍기능 구현 목록
 

## 🎯기능 구현 흐름
```
- 로또 구입 금액을 입력받는다.
- 당첨번호를 입력받는다. 
- 보너스 번호를 입력받는다. 
- 발행한 로또 수량을 출력한다. 
- 발행한 로또 번호를 전부 출력한다. 
- 당첨 내역을 출력한다. 
- 수익률을 출력한다. 
```
## 🎯기능 목록 및 세부 설정

### 1️⃣ 로또 구입 금액 입력 
```
- 로또 금액을 입력받는다. 
    - 로또 1장은 1000원 이므로 입력받은 금액이 1000 단위인지 확인한다.  
-----------------------------------------------  
🚨 1000원으로 나누어 떨어지지 않는 경우 예외처리한다.
🚨 글자를 받을 경우 예외처리한다. 
```

### 2️⃣ 개수만큼 로또 발행 및 출력
```
- 랜덤메소드를 사용해서 로또를 1번에서 받은 로또 개수만큼 생성한다.
    - 로또 객체를 생성한다.
    - 로또 번호는 서로 중복되면 안된다.
    - 로또 번호의 개수는 6개이다.
    - 로또 숫자들은 1 ~ 45 사이의 숫자이다. 
- 만들고 난 후에 로또 번호를 출력한다.
    - 오름차순으로 숫자를 나열해야 한다.
-----------------------------------------------
🚨 로또 번호는 다음과 같은 기준을 정하고 지키지 못하면 예외처리한다.
    - 로또 번호가 서로 중복되는 경우 예외 처리한다. 
    - 로또 번호가 6개를 넘거나 부족한 경우 예외 처리한다. 
    - 로또 번호가 1 ~ 45 사이의 번호가 아닐 경우 예외 처리한다.  
```



### 3️⃣ 당첨번호 입력
```
-  당첨번호를 입력한다.
    - 당첨번호는 쉼표(,)로 구분한다.
    - 당첨번호 개수는 로또 번호 개수와 일치해야 하므로 6개만 받는다.
--------------------------------------------------
🚨 로또 번호는 다음과 같은 기준을 정하고 지키지 못하면 예외처리한다. 
    - 로또 번호가 서로 중복되는 경우 예외 처리한다. 
    - 로또 번호가 6개를 넘거나 부족한 경우 예외 처리한다. 
    - 로또 번호가 1 ~ 45 사이의 번호가 아닐 경우 예외 처리한다. 
```
### 4️⃣보너스 번호 입력
```
- 보너스 번호를 입력한다. 
--------------------------------------------------
🚨 1 ~ 45 사이의 숫자가 아닐경우 예외처리한다.
🚨 당첨번호와 중복될 경우 예외처리한다. 
```


### 5️⃣로또 매칭 
```
- 로또 한 객체에 있는 번호와 당첨 번호를 비교한다.
- 당첨 번호를 가지고 있을 경우 로또 객체의 countOfWinningNum 에  +1 한다.
```
### 6️⃣당첨 내역 출력 
```
- 주어진 형태에 맞춰서 당첨 내역을 출력한다. 
    - 3개 일치 (5,000원) - n개
    - 4개 일치 (50,000원) - n개
    - 5개 일치 (1,500,000원) - n개
    - 5개 일치, 보너스 볼 일치 (30,000,000원) - n개
    - 6개 일치 (2,000,000,000원) - n개
```
### 7️⃣ 수익률 출력
```
- 수익률을 출력한다.
    - 수익률을 계산한다.
    - (1-(각 일치 개수에 따른 상품금액 총합을 로또 금액으로 나눈 값)) x 100 실행
    - 수익률은 소수점 둘쨰자리에서 반올림한다. 
--------------------------------------------------
🚨 얻은 금액이 0 일 경우 "수익을 얻지 못하였습니다." 출력
```
