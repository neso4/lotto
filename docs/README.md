# 미션 - 로또

## 💫 구현 기능 목록

### 1. 로또 구입 금액 입력

- 숫자 검증
- 단위 검증

### 2. 로또 발행 결과 출력

- 로또 수량 계산
- 로또 번호 생성
- 오름차순 정렬
- 구매 수량 출력
- 로또 발행 결과 출력

### 3. 당첨 번호 입력

- 숫자 범위 검증
- 숫자 개수 검증
- 중복 검증

### 4. 보너스 번호 입력

- 숫자 범위 검증
- 중복 검증

### 5. 당첨 통계 출력

- 당첨 개수, 금액 계산
- 당첨 내역 출력

### 6. 수익률 출력

- 수익률 계산
- 수익률 출력

> *검증 실패 -> IllegalArgumentException 발생 -> Error 메시지 출력 후 재입력
