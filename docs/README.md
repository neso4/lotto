# 💰 로또
* * *

## 📋 구현할 기능 목록
* * *


### 1. 구입 금액 입력받기
- 입력 값 유효성 판단
- 양의 정수
- 10만원 이내

### 2. 구매한 로또 출력
- 구입 개수 출력
- 로또 숫자 출력

### 3. 당첨 번호 입력받기
- 입력 값 유효성 판단
- 개수 확인(6개)
- 당첨 번호 범위 확인
- 당첨 번호 중복 확인

### 4. 보너스 번호 입력받기
- 입력 값 유효성 판단
- 보너스 값 범위 확인
- 보너스 번호와 당첨 번호가 중복되지 않는지 확인

### 5. 당첨 내역 출력
- 각 일치 개수에 해당하는 조건과 개수 출력

### 6. 수익률 출력
- 수익률 계산 및 출력(소수점 1자리 수까지 출력)
