
# 흐름별 정리

- 로또 기능 정의

## 구입금액 입력

### 입력값 검증

- 빈값 검증
- 중복값 검증
- 숫자 검증
  - 1000원 단위 검증
  - 범위 검증

## 구매 완료시 완료 메세지 출력

## 당첨 번호 입력

### 번호입력

- 숫자 1~45 로 구분된 6자리 수
- 구분자 `,`

### 보너스 번호 입력

- 1~45 숫자

## 구매 로직

### 구매한 로또 번호 생성

- 로또 번호 6자리 int배열 생성 (1~45 숫자)
- 오름차순 정렬

### 구매한 로또 번호 출력

# 당첨 통계

- 일치 출력
- N개 일치 ()
- 수익률
  - 수익률 계산
  - 수익률 출력

- 예외 처리
- [x] 공통 사용자 입력 예외
- [x] 구매 금액 예외
- [x] 로또 번호 예외
- [x] 당첨 번호 예외
- [x] 보너스 번호 예외

# 기능별 정의

- 메세지 출력
- 검증
- 로또 숫자 집합
  - 로또 숫자
    - 1~45 사이의 수
  - 6개의 로또 숫자 + 보너스 숫자 1개
- 당첨 검증
  - 6개 일치
