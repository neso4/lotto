- [o] 로또 구입 금액 입력 받기
   - 구입 금액 입력 시 해당하는 만큼 로또 발행
   - 구입 금액 1,000원 단위로 입력 받음
   - 1,000원으로 나누어 떨어지지 않는 경우 예외 처리
   - 구입할 수 있는 로또 개수 반환
- [o]로또 랜덤 숫자 발행
   - 중복되지 않는 6개 숫자를 통해 로또 발행
   - 당첨 번호 추첨, 중복되지 않은 숫자 6개 ,보너스 번호 1개
   - 로또 번호 오름차순 정렬
- [o]당첨 번호 입력 받기
    - 쉼표(,) 기준 구분
    - 보너스 번호입력
