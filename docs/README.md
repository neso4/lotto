- 로또 1개당 천원

1. 구입 금액 입력하여 몇개 구매하는지 출력
   - 구입한 개수만큼 로또 객체 생성
2. 구매한 전체 로또 번호 출력
   - 로또 객체 리스트 반환
3. 당첨 번호 및 보너스 번호 입력
4. 입력한 번호와 내가 구매한 번호와 일치 확인해 각각 출력
   - 1등: 6개 번호 일치 / 2,000,000,000원
   - 2등: 5개 번호 + 보너스 번호 일치 / 30,000,000원
   - 3등: 5개 번호 일치 / 1,500,000원
   - 4등: 4개 번호 일치 / 50,000원
   - 5등: 3개 번호 일치 / 5,000원
5. 수익률 계산
