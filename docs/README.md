기능 구현 목록

- 예외 발생 시 [ERROR]로 시작하는 예외 처리 후 다시 입력받기

- 구매할 로또 금액 입력받기(1000원 단위)
- 1000원 단위가 아닐 경우 예외 처리

- 구매한 로또 장수 출력
- 구매한 장수만큼 로또 번호 생성 (1~45까지의 6자리 수)
  
- 당첨 번호 입력받기
- 숫자 범위, 숫자 개수 맞지 않을 경우 예외 처리
- 중복된 숫자 있을 경우 예외 처리
  
- 보너스 번호 입력받기
- 당첨 번호와 중복될 경우 예외 처리

- 당첨 통계 출력하기
- 생성한 로또 번호와 사용자 입력 번호가 몇 개나 겹치는지 판단
- 수익률 계산 ((5000 / 8000) * 100%)
