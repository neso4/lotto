
# 로또 게임

## 기능 목록
- [x] 로또 구입 금액을 입력받는다.
- [x] 입력받은 금액 갯수만큼 로또를 발행한다.
  - [x] 로또 번호는 1부터 45까지의 임의의 서로 다른 6개의 수로 구성된다.
  - [x] 6자리 아니면 예외처리
  - [x] 번호는 오름차순으로 정렬한다.
  - [x] 1000원으로 나누어 떨어지지 않는 경우 예외 처리.
- [x] 당첨 번호와 보너스 번호를 입력받는다.
- [x] 발행된 로또 티켓들과 당첨 번호를 비교한다.
  - [x] 당첨 순위와 금액을 파악한다.
  - [x] 각 등수별 당첨 횟수를 저장한다.
  - [x] 구입 금액을 바탕으로 수익률을 계산한다.
