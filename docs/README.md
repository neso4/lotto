# 기능목록
- [V] 로또 구입 금액 입력
- [V] 구입 금액 예외 처리
    - [V] 빈 입력
    - [V] 숫자
    - [V] 자연수
    - [V] 1000단위
- [V] 구입 급액으로 구입 갯수 계산
- [V] 구입 갯수만큼 로또 번호 생성
- [V] 생성한 로또 번호 출력
- [V] 당첨 번호 입력
- [V] 당첨 번호 예외 처리
    - [V] 빈 입력
    - [V] 쉼표로 분리
    - [V] 6개 확인
    - [V] 숫자 범위 1 ~ 45 확인
    - [V] 중복 숫자 확인
- [V] 보너스 번호 입력
    - [V] 빈 입력
    - [V] 숫자
    - [V] 자연수
    - [V] 숫자 범위 1 ~ 45 확인
    - [V] 당첨 번호와 중복 숫자 확인
- [V] 일치 숫자 갯수 확인
- [V] 보너스 번호 일치 확인
- [V] 당첨금 계산
- [V] 수익률 계산
- [V] 에러 문구 출력
