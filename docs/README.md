### 로또

> LottoController Class 
> 전체 게임을 컨트롤 하는 컨트롤러
> 
> playLotto() : 시작할 때 실행되는 함수 <br>
>   > start() : 구매금액을 입력받고 적절한 입력값인지 판단<br>
>   > generateLottoNumber : 로또 번호를 생성 <br>
>   > middle() : 당첨번호들을 입력 <br>
>   > calculate() : 당첨결과 계산 <br>
>   > printEnd() : 당첨 통계 출력 <br>
