-  로또 판매 기능 작성
  - [ ] "구입금액을 입력해 주세요." 문구를 출력한 후 금액을 입력받는다.
  - [ ] 금액에 대해 유효성을 검사한다.
    - [ ] 천원 단위의 입력 외에는 예외가 발생한다.
  - [ ] 입력값/1000 만큼의 로또를 생성한다.
    - [ ] 생성한 로또 번호를 알려준다.
  - [ ] 컴마 (,) 를 구분자로 당첨 번호를 입력받는다.
  - [ ] 보너스 번호를 입력 받는다.
  - [ ] 당첨된 로또에 대한 통계를 알려준다.
    - [ ] 3개 이상 맞은 로또부터 몇 개가 맞았는지 알려준다.
    - [ ] 당첨된 금액과 구매 금액으로 수익률을 알려준다.
