# 예외사항

  
---  

### 구입금액 입력

- [X] 숫자만 포함되어있는지
- [X] 1000의 배수인지
- [X] 너무 크지 않은 숫자인지 (Integer 범위)

### 당첨번호 입력

- [X] `,`로 나누었을 때, 숫자만 포함되어있는 지
- [X] `,`로 나누었을 때, 모든 숫자가 1 ~ 45 인지
- [X] `,`로 나누었을 때, 중복이 없는 지
- [X] `,`로 나누었을 때, 길이가 6인지
- [X] `,`가 연속해서 나올 경우

### 보너스 번호 입력

- [X] 숫자인지
- [X] 1 ~ 45 인지
- [X] 이미 당첨번호에 포함이 되어있는 지
