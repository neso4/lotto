## 프로그램 흐름

- 로또를 구매 한다
    - 구입 금액을 입력함
    - 구매한 로또를 출력함
- 당첨 번호를 입력한다.
    - 당첨 번호는 6자리를, 보너스 번호는 1자리를 입력
- 결과를 출력한다.
    - 당첨 통계
        - n개 일치 (%d원) - %d개
    - 수익률
        - 총 수익률은 %f원 입니다.
