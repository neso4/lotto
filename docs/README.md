## 기능 구현 사항 정리
1. 로또 금액 입력 받기  
   2.  1000 단위로 입력 받는지 확인 및 예외 처리  
   3.  숫자가 아닐 경우 예외 처리  
4. 로또 번호 생성  
   5. 범위는 1~45  
   6. 중복되지 않은 6개의 숫자 생성  
   7. 당첨 번호 입력 받을 시 보너스 번호 1개 추가로 더 받기  
4. 금액에 맞게 로또 수량만큼 번호 생성  
   5. 당첨 번호와 비교하여 당첨 내역 수렴하기  
   6. 수익율 계산  

## 주의사항   
1. 에러 메세지 추가  
2. 함수의 길이가 15라인 넘어가지 않도록 하기  
3. else 사용하지 않기  
4. Java Enum 적용하기  
5. 도메인 로직 단위 테스트 구현
