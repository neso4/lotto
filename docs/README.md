# 미션 - 로또

## 구현 기능

* __입력__

- [x] 로또 구입 금액을 입력 받는다.
- [x] 당첨 번호를 입력 받는다.
- [x] 보너스 번호를 입력 받는다.

* __로또 게임__

- [ ] 당첨 번호와 보너스 번호를 생성한다.
- [ ] 당첨된 로또가 있는지 확인한다.
- [ ] 수익률을 계산한다.

* __출력__

- [x] 발행한 로또 수량 및 번호를 출력한다. 로또 번호는 오름차순으로 정렬하여 보여준다.
- [x] 당첨 내역을 출력한다.
- [x] 수익률을 출력한다.
- [ ] 예외 상황에 맞는 예외를 출력한다.

***

## 추가 예외 상황

* __입력__

- [ ] 로또 구입 금액이 1,000원 단위로 나누어 떨어지지 않는 경우
- [ ] 로또 구입 금액이 1,000 ~ Integer.MAXVALUE 사이의 범위를 벗어난 경우
- [ ] 당첨 번호가 쉼표(,)를 기준으로 구분되지 않는 경우
- [ ] 당첨 번호에 중복이 존재하는 경우
- [ ] 보너스 번호가 당첨 번호와 중복된 경우
