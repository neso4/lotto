# java-lotto-6 기능 요구 명세서

## 주요 기능
사용자가 구입할 로또의 값을 입력하면 이를 1000으로 나누어 발행할 로또 개수를 정한다.
 이후 로또는 총 6개로 이루어진 로또번호(1~45)를 중복 없이 발행한다. 
이후 사용자는 당첨번호 6개(1~45)를 콤마로 구분하여 입력하고 보너스 번호(1~45)도 입력한다.
사용자가 입력한 로또 당첨 번호와 보너스 번호를 바탕으로 당첨 금액을 계산하고 입력한 금액과 당첨금액을 비교하여 총 수익률을 계산한다.

## 1. 사용자 입력

- ### 로또 구입 금액 입력
  +  예외처리 목록
  +  [X] null 입력값에 대한 예외처리 
  +  [X] 양수가 아닌 정수 입력에 대한 예외처리
  +  [X] 1000으로 나누어 떨어지지 않는 수에 대한 예외처리
+ ### 로또 당첨 및 보너스 번호 입력
  +  [X] null 입력값에 대한 예외처리
  +  [X] 정수가 아닌 입력에 대한 예외처리
  +  [X] 입력값 범위에 때란 예외처리 (1~45)
  +  [X] 중복된 입력 값에 대한 예외처리
  +  [X] 로또 당첨 번호 및 보너스 번호 입력 값 개수에 대한 예외처리
<br>

## 2. 로또 번호 생성 및 출력
- ### 사용자가 입력한 로또 금액을 바탕으로 총 발행 로또의 수를 구한다.
- ### 발행한 로또의 수 만큼 중복되지 않은 6개의 로또 번호(1~45)를 출력한다.
<br>

## 3. 수익률 계산 및 출력
 - ### 각각의 로또 번호와 입력한 로또 당첨 번호를 비교하여 중복되는 값의 개수를 구한다.
 - ### 각각의 로또 번호와 입력한 보너스 번호를 비교하여 중복되는 값의 개수를 구한다.
 - ### 중복된 값의 개수와 보너스 값 중복 여부를 확인하여 전체적인 당첨 금액을 계산한다.
 - ### 당첨금액과 입력한 구입 금액을 비교하여 수익률을 계산하여 출력한다.
