# 🎯 구현 기능 목록
- [ ] 사용자에게 로또 구입 금액 입력받기
- [ ] 로또 구입 금액 만큼 로또 발행하기(중복되지 않는 6개의 숫자)
- [ ] 당첨번호와 보너스 번호를 생성한다
- [ ] 발행된 로또의 당첨 여부를 확인한다
    ````
    당첨 조건)
    1등 : 6개 번호 일치
    2등 : 5개 번호 + 보너스 번호 일치
    3등 : 5개 번호 일치
    4등 : 4개 번호 일치
    5등 : 3개 번호 일치 
    ````
###  🚫예외처리

- [ ] 구매 금액이 숫자가 아닌 경우
- [ ] 구매 금액이 1000원 단위가 아닌 경우
- [ ] 구매 금액이 10만원 이상일 경우
