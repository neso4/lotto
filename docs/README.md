# 로또 번호 생성 및 당첨 확인 시스템

## 프로젝트 개요
이 프로젝트는 사용자로부터 희망하는 로또 구매 금액을 입력 받아 자동으로 로또 번호를 발급하는 어플리케이션입니다. 당첨 번호와 보너스 번호를 입력하면, 자동 발매된 로또 결과와 비교하여 당첨 여부를 확인할 수 있습니다.

## 사용 방법
1. 구매 금액 입력: 사용자는 구매하고자 하는 로또의 총 금액을 입력합니다.
2. 로또 번호 출력: 시스템은 자동으로 로또 번호를 생성하고 화면에 표시합니다.
3. 당첨 번호 입력: 사용자는 당첨 번호를 쉼표로 구분하여 입력합니다.
4. 결과 확인: 시스템은 당첨 결과와 수익률을 계산하여 화면에 표시합니다.


## 세부 기능
### **✅ 로또 번호 생성**

- 지정 범위 내에서 중복되지 않는 번호 생성
  - 생성된 번호의 범위 검증
  - 번호 리스트의 중복 값 검증
  - 번호 리스트 길이 검증
- 번호 리스트 오름차순 정렬

### **✅ 입력 및 구매 관리**

- 구입 금액에 따른 로또 발행
- 구입한 로또 개수 출력
- 구매한 로또 번호 출력

### **✅ 예외 처리**

- 금액 입력 형식 검증
- 당첨 번호 입력 형식 검증
- 당첨 번호의 범위 및 중복 검증
- 보너스 번호의 중복 및 범위 검증
- 입력 오류 후 재입력 요청

### **✅ 당첨 번호 및 결과 처리**

- 당첨 번호와 보너스 번호 추첨
- 당첨 통계 출력
  - 구매한 로또 번호와 당첨 번호 비교
  - 당첨 등수 및 금액 계산
- 수익률 출력 (소수점 둘째자리 반올림)
  - 비교한 결과에 따른 수익률 계산
  - 수익률 출력
