# 기능 요구 사항
## 에러 처리
- 잘 못된 입력에 대해서 IllegalArgumentException 발생
- 에러 메세지는 "[ERROR]" 로 시작
- 에러 출력후 다시 입력을 받는다

## 입력
### 로또 구입 금액
- 구입 금액은 1,000 단위로 입력 가능
- 아닐 경우 에러 처리

### 당첨 번호 입력
- 중복 없는 1 ~ 45 사이 자연수 6개 를 입력 받습니다.
- 각 숫자 구분은 "," 로 합니다
- 입력 예시
    - 1,2,3,4,5,6
- 아닐 경우 에러 처리

### 보너스 번호 입력
- 당첨 번호에 없는 1 ~ 45 사이에 자연수 1개를 입력 받습니다.
- 아닐 경우 에러 처리

## 구입한 로또 생성
- 각 로또는 1 ~ 45 사이에 중복없는 자연수 6 개로 이루어져 있다
- 구입한 로또 개수 출력
- 각 로또의 구성 숫자 출력
