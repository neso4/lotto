# 로또 게임 기능 목록

## 기본 요구 사항

- [ ] 로또 번호의 숫자 범위는 1~45
- [ ] 중복되지 않는 6개의 숫자를 뽑는 로또 발행 기능
- [ ] 중복되지 않는 숫자 6개와 보너스 번호 1개를 뽑는 당첨 번호 추첨 기능
- [ ] 당첨 기준과 금액에 따른 등수 분류 기능

## 구입 및 발행 기능

- [ ] 로또 구입 금액 입력 및 예외 처리 기능
- [ ] 입력 받은 금액에 해당하는 로또 발행 기능
- [ ] 발행된 로또 수량 및 번호 출력 기능

## 당첨 번호 입력 및 검증 기능

- [ ] 당첨 번호 입력 기능
- [ ] 보너스 번호 입력 기능
- [ ] 입력된 번호의 유효성 검증 기능

## 당첨 결과 처리 기능

- [ ] 사용자의 로또 번호와 당첨 번호 비교 기능
- [ ] 당첨 내역 계산 및 출력 기능
- [ ] 수익률 계산 및 출력 기능

## 예외 처리 및 유효성 검사

- [ ] 잘못된 값 입력 시 IllegalArgumentException 발생 기능
- [ ] 오류 메시지 출력 및 재입력 요청 기능
- [ ] 명확한 유형의 예외 처리 기능

## 입출력 요구 사항

- [ ] 로또 구입 금액 입력
- [ ] 당첨 번호 입력
- [ ] 보너스 번호 입력
- [ ] 발행된 로또 번호 출력
- [ ] 당첨 내역 출력
- [ ] 수익률 출력
- [ ] 예외 상황 시 에러 문구 출력
