# 프로젝트 소개

# 주요 상세 기능

#### 1. 사용자에게 구매 금액을 입력받고 해당하는 개수만큼 로또 발행
    - 추가로 당첨 번호와 보너스 번호를 입력
#### 2. 사용자가 구매한 로또 번호와 당첨 번호를 비교
    - 당첨 내역 출력 및 총 수익률 출력
#### 3. 예외 상황에 대해 처리 및 에러 문구 출력
