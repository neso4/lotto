---

### 기능 구현

---
### I/O
```
- 콘솔을 통한 문자열 입력 기능
- 콘솔을 통한 메시지 출력 기능
```

### Validator
```
- 구입 금액 검증
  - 입력 값이 숫자인지 확인합니다.
  - 입력 값이 1000으로 나누어 떨어지는지 확인합니다.
  - 입력 값이 0보다 크고 구매 제한 금액보다 작은지 확인합니다.
- 당첨 번호 검증
  - 입력 값을 ,로 구분하여 분리합니다.
    - 당첨 번호
  - 숫자 중복 확인
    - 당첨 번호
    - 보너스 번호
  - 숫자가 1~45 사이의 값인지 확인합니다.
    - 당첨 번호
    - 보너스 번호
```
### Lotto
```
- 당첨 번호와 일치하는지 확인합니다.
```
### Game
```
- 메시지 생성
  - 구매 목록 메시지를 생성합니다.
  - 결과 메시지를 생성합니다.
- 게임
  - 구입 금액을 입력받아 복권을 발행합니다.
  - 당첨 번호를 입력받아 저장합니다.
  - 게임 결과를 확인합니다.
  - 수익률을 확인합니다.
```
### Exception
```
- 에러 메시지 생성
```
