## 기능 목록

- 로또 클래스에 대한 검증 로직을 구현한다.
    - 6개의 숫자인지 판단한다.
    - 하나의 숫자에 대해서 1~45사이의 숫자인지 판단한다.
    - 6개의 숫자가 1~45 사이의 숫자인지 판단한다.
    - 모두 중복되지 않은 6개의 숫자인지 판단한다.
- 보너스 번호 하나에 대해서 판단한다.
    - 기존에 뽑은 6개의 숫자와 중복되지 않는다.
    - 1~45 사이의 숫자이다.

- 위의 조건을 만족하는 6자리 로또를 하나 발행한다.
    - `camp.nextstep.edu.missionutils`에서 제공하는 `Randoms`를 사용해서 구현한다.
    - 형식에 맞춰 오름차순으로 정렬하여 출력한다.
- 로또 번호를 발행하여 진행하는 게임 기능을 구현한다.
    - 로또를 발행한다.
    - 당첨 통계를 낸다.
        - 3개부터 6개까지 순서대로 몇 개의 번호가 일치하는지 판단한다.
        - 당첨 금액의 총 합을 계산한다.
        - 수익률을 계산하여 출력한다.(수익률은 소수점 둘째 자리에서 반올림하고 소수점 첫째 자리는 빈자리이면 0으로 채운다.)

#### 입출력

- 구입 금액을 입력받는다.
- 해당 개수만큼 로또를 생성한다.
- 구입 금액에 해당하는 로또를 생성하여 출력한다.
- 당첨 번호를 입력받는다.
- 당첨 통계를 낸다.
