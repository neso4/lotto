### 구현사항
- 로또 구입 금액을 입력 받는다.
  - 1000원 단위로 입력받는다.
  - 1000원으로 나누어 떨어지지 않는 경우 예외를 발생하고 입력을 다시 받는다.
  - 로또 번호를 생성한다.
    - 6자리 숫자여야 한다.
    - 중복된 숫자가 있으면 안된다.
    - 오름차 순으로 정렬한다.
- 당첨 번호를 입력받는다.
  - 6자리 숫자여야 한다. 
  - 중복된 숫자가 있으면 안된다.
  - 예외 발생시 입력을 다시 받는다.
- 보너스 번호를 입력받는다.
- 당첨내역을 출력한다.
  - 각 등수별 로또의 개수를 출력한다.
- 수익률을 출력한다.
