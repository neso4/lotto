## 기능 구현 목록

기능 목록

- 로또를 구매 한다
    - 구입 금액을 입력
        - 음수, 0, 또는 1,000으로 나누어 떨어지지 않는 값을 입력하면 예외 발생
    - 구매한 로또를 출력함
- 당첨 번호를 입력한다.
    - 당첨 번호는 6자리를, 보너스 번호는 1자리를 입력
        - 당첨 번호가 서로 중복될 경우 예외 발생
        - 보너스 번호가 당첨 번호와 중복될 경우 예외 발생
        - 로또 숫자들이 정해진 범위 (1 ~ 45)를 벗어나면 예외 발생
- 결과를 출력한다.
    - 당첨 통계
        - n개 일치 (%d원) - %d개
    - 수익률
        - 총 수익률은 %f원 입니다.
