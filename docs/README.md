## 구현기능
1. 사용자 입력
- 로또 구입 금액 입력
- 구입 금액 검사 (1000원 단위)
- 구입 금액만큼 로또 발행
- 로또번호(무작위값) 6자리 검사
- 로또번호 숫자 범위(1~45) 검사
- 로또번호 중복 검사
- 로또번호 오름차순 정렬
- 로또 수량 및 번호 출력


2. 로또 추첨 시작
- 당첨번호 입력
- 당첨번호 입력(","로 구분)
- 당첨번호 6자리 검사
- 당첨번호 숫자 범위(1~45) 검사
- 당첨번호 중복 검사
- 보너스번호 입력
- 보너스번호 1자리 검사
- 보너스번호 숫자 범위(1~45) 검사


3. 평가
- 구매한 로또번호랑 당첨 번호 비교
- 당첨 내역 출력
- 수익률 계산 (소수점 둘째자리 반올림)


4. 사용자 입력 예외처리
- IllegalArgumentException 발생
- 에러 메세지 출력
- 입력 재실행
