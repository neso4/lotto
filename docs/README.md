<h1>구현할 기능 목록</h1>
<ol>
    <li>랜덤으로 숫자를 제공해 주는 기능</li>
    <li>당첨 내역을 알려주는 기능</li>
    <li>총 수익률을 알려주는 기능</li>
    <li>예외를 알려주는 기능</li>
</ol>
<h1>정리한 기능 목록</h1>
<ol>
    <li>getLotto</li>
    1에서 45사이의 6개의 숫자를 받는 메소드
    <li>printLotto</li>
    6개의 받은 숫자를 정렬을 해서 보여주는 메소드
    <li>winLotto</li>
    입력한 숫자를 이용해 몇개의 숫자를 맞췄는지 확인하는 메소드
    <li>winPrice</li>
    몇개를 맞췄는지를 이용해 얼마를 받을 수 있는지와 수익률을 확인하는 메소드
    <li>getMon</li>
    구입금액을 받는 메소드
    <li>getNum</li>
    사용자의 숫자를 받는 메소드
    <li>checkBonus</li>
    사용자의 보너스 숫자를 받는 메소드
</ol>
