# 로또

## 📍 구현 기능 목록

***

**1. 구입 금액 입력**

- 예외 [ERROR]
    - 1000원 단위로 나누어 떨어지지 않은 경우
    - 숫자가 아닌 (ex.문자열)을 입력한 경우
    - 공백을 입력한 경우

**2. 당첨 번호 입력**

- 예외 [ERROR]
    - 범위(1~45)를 벗어난 경우
    - 6개의 숫자로 이루어지지 않은 경우
    - 쉼표(,)로 구분되지 않은 경우
    - 숫자가 아닌 (ex.문자열)을 입력한 경우
    - 공백을 입력한 경우

**3. 보너스 번호 입력**

- 예외 [ERROR]
    - 범위(1~45)를 벗어난 경우
    - 1개를 초과하여 입력한 경우
    - 숫자가 아닌 (ex.문자열)을 입력한 경우
    - 공백을 입력한 경우

**4. 발행한 로또 수량 및 번호 출력**

- 로또 번호 - 오름차순으로 정렬

**5. 당첨 내역 출력**

- 당첨 등수 - 오름차순으로 정렬
- 수익률 - 소수점 둘째짜리에서 반올림
