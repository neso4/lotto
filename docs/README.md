# 구현할 기능

## 입출력

### 입력

- 사용자로부터 구입 금액을 입력받는다
  - [x] 구입 금액은 1000의 배수이다
  - [x] 정수만 입력받는다

- 사용자로부터 로또 번호를 입력받는다.
  - [x] 숫자 6개를 입력받는다.
  - [x] `,`를 기준으로 숫자를 구분한다.
  - [x] `1` ~ `45` 사이의 숫자를 입력받는다.
  - [x] 정수만 입력받는다.
  - [x] 숫자는 중복을 허용하지 않는다.

- 사용자로부터 보너스 번호를 입력받는다.
  - [x] 사전에 입력받은 6개 숫자 중 입력되지 않은 숫자를 입력받는다.
  - [x] `1` ~ `45` 사이의 숫자를 입력받는다.

- [x] 올바르지 않은 입력을 받으면 `IllegalArgumentException`을 발생시키고, 잘못 받은 부분부터 다시 입력을 받는다.

### 출력

- 사용자로부터 입력받음 금액 / 1000 번 만큼 랜덤한 로또를 출력한다.
  - [x] `[` `]` 사이에 6자리 숫자를 `,`로 구분한다.
  - [x] 한 줄에 하나의 로또를 출력한다.

- 입력에 따른 당첨 통계를 출력한다.
  - [x] `당첨 통계` 를 출력한 다음 결과를 알려준다.
  - [x] `---` 다음 줄 부터 결과를 출력한다.
  - [x] 일치하는 결과를 출력해준다.
    - [x] `n개 일치 (m원) - l개` 의 형식으로 출력한다.
- [x] 총 수익률을 출력한다.
  - [x] `총 수익률은 ld% 입니다` 의 형식으로 출력한다.
  - [x] 수익률은 소수점 둘째 자리에서 반올림한다.

- [x] 예외 메세지는 `[ERROR]` 로 시작한다.

## Lotto

- [x] 6자리 숫자로 구성된다.
- [x] 숫자간 중복을 허용하지 않는다.
- [x] 숫자는 `1` ~ `45` 사이의 숫자이다.
- [x] 숫자는 오름차순으로 정렬되어 있다.

## 게임

- [x] 사용자에게 금액을 입력받는다.
- [x] 금액만큼 로또를 보여준다.
- [x] 사용자에게 로또 번호를 입력받는다.
- [x] 보너스 번호도 입력받는다.
- [x] 로또에 대한 당첨 통계 및 수익률을 출력한다.

## Judge

- [x] 당첨 번호와 로또를 비교해 순위를 매긴다.
- [x] 순위 정보 및 당첨금 정보를 저장한다.
- [x] 수익률을 계산한다.

## RandomGenerator

- [x] 난수 생성기를 통해 6자리 로또 번호를 생성한다.
