1. 기능 요약

(1) 로또 번호의 숫자 범위는 1~45
</br>
(2) 중복 없는 6개의 무작위 숫자
</br>
(3) 당첨 번호 추첨 시 -> 중복 없는 숫자 6개 + 보너스 번호 1개 ( 사용자가 입력 )
</br>
(4) 당첨은 1등 ~ 5등까지
- 1등 : 번호 6개 일치 / 20억
- 2등 : 번호 5개 일치 + 보너스 번호 일치 / 3천만
- 3등 : 번호 5개 일치 / 150만
- 4등 : 번호 4개 일치 / 5만
- 5등 : 번호 3개 일치 / 5천

(5) 구입 금액을 입력 -> 그만큼 로또를 발행
</br>
(6) 로또 1장은 1000원
</br>
(7) 당첨 번호 + 보너스 번호를 입력 받음
</br>
(8) 사용자 입력과 무작위 생성된 로또 번호 비교 -> 당첨 내역 / 수익률 출력 -> 종료
</br>
(9) 값 입력 에러 처리
