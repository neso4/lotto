# 로또 구현할 기능 목록

## 1. 로또 번호 생성

- [X] 구입 금액을 입력받는다.
- [X] 구입 금액을 기반으로 로또 번호를 생성한다.

## 2. 로또 번호 출력

- [X] 구입 금액을 기반으로 구매한 장수를 출력한다.
- [X] 생성된 로또 번호를 출력한다.

## 3. 당첨 번호 및 보너스 번호 입력

- [X] 당첨 번호를 입력받는다.
- [X] 보너스 번호를 입력받는다.

## 4. 로또 번호와 당첨 번호 비교

- [X] 각 로또 번호와 당첨 번호를 비교하여 일치하는 숫자의 개수를 센다.
- [X] 당첨 기준에 따라 수익률을 계산한다.

## 5. 당첨 내용 출력

- [X] 당첨 기준에 따라 일치하는 개수와 개수별 상금을 출력한다.
- [X] 총수익률을 계산하여 출력한다.

## 6. 에러 처리

- [X] 입력값이 유효한 범위인지 확인한다.
